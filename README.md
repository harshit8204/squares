## 💡 About

**Squares** is a CSS-first creative coding project — an animated visual experiment featuring squares brought to life entirely through **CSS3 keyframes, transforms, and transitions**.

> ✏️ *Update this line with a one-sentence description — e.g. "Infinitely looping animated squares with rotating gradients and staggered timing" or "A grid of colour-shifting squares animated with CSS keyframes and custom properties."*

With **CSS at 64.5%** of the codebase, this project is a focused deep-dive into the creative and technical limits of CSS animation — no JavaScript event loops, no canvas, no animation libraries. Just stylesheets doing the heavy lifting.

---

## 🛠️ Tech Stack

| Technology | Role | Share |
|------------|------|-------|
| 🎨 **CSS3** | All animations, transforms, colours, layout | 64.5% |
| 🌐 **HTML5** | Semantic structure & square elements | 35.5% |
| ⚡ **JavaScript** | — *Not used* — | 0% |

> **CSS is the star of the show here.** 64.5% CSS with zero JavaScript is a bold statement —
> and a genuine showcase of advanced stylesheet skills.

---

## ✨ Features

- 🟦 **Animated squares** — smooth, continuous CSS-driven motion
- 🎞️ **CSS keyframe animations** — `@keyframes` for fluid, looping effects
- 🔄 **Transform effects** — `rotate`, `scale`, `translate` in pure CSS
- ⏱️ **Staggered timing** — `animation-delay` for cascading visual rhythm
- 🌈 **Colour transitions** — gradient shifts and colour cycling via CSS
- 📱 **Responsive layout** — adapts across all screen sizes
- ⚡ **Instant load** — no JS parsing, no network requests, blazing fast
- ♿ **Accessible markup** — clean, semantic HTML structure

## 🚀 Getting Started

No setup. No install. No terminal commands needed.

### Option 1 — Clone & Open

```bash
# 1. Clone the repository
git clone https://github.com/harshit8204/squares.git

# 2. Navigate into the folder
cd squares

# 3. Open in your browser
open index.html         # macOS
start index.html        # Windows
xdg-open index.html     # Linux
```

### Option 2 — Live Server (Best for Dev)

Using **VS Code**? Install [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer):

1. Right-click `index.html`
2. Click **"Open with Live Server"**
3. See changes instantly as you tweak CSS ✅

## 🧠 What This Demonstrates

This project highlights a set of **advanced CSS skills** that many developers never fully explore:

| CSS Technique | What It Shows |
|---------------|--------------|
| 🎞️ **`@keyframes`** | Custom multi-step animation sequences |
| 🔄 **`transform`** | 2D/3D rotation, scaling, and translation |
| ⏱️ **`animation-delay`** | Staggering elements for cascading effects |
| ⚡ **`animation-timing-function`** | `ease`, `linear`, `cubic-bezier` mastery |
| 🔁 **`animation-iteration-count`** | Infinite looping and controlled repetition |
| 🌈 **CSS gradients** | `linear-gradient` / `radial-gradient` for colour |
| 📐 **CSS Grid / Flexbox** | Clean, responsive square layouts |
| 🎨 **CSS custom properties** | `--variables` for maintainable theming |

> For UI/UX, frontend, and creative developer roles, a pure CSS animation project
> signals you can build performant, smooth experiences without reaching for JavaScript.

## 🌍 Deployment
